# go_socket
 
学校计网的要求就用刚学会的golang语言写了一个socket的收发代码，只有关于**TCP**的部分，UDP稍等更新

使用方法 
```shell
(Windows)
cd /src/server
go mod tidy
go build
server.exe
cd ..
cd client
go mod tidy 
go build
client.exe

(Linux)
cd /src/server
go mod tidy
go build
./server
cd ..
cd client
go mod tidy
go build
./client
```
